HTML5 使用 Cordova 混合编译 - 环境配置篇

安装Cordova

 yarn add global cordova

依赖环境安装

事先确保本机已安装 JDK、SDK 并配置好 JAVA_HOME、 ANDROID_HOME 环境 和 Node.js

  1. JDK地址安装

JDK下载

注意：勾选Accept License Agreement、选择是32位还是64位操作系统

  1. JDK环境变量配置

配置环境变量：

右击 我的电脑 > 属性 > 高级系统设置 > 高级 > 环境变量

    - 在系统变量里新建`JAVA_HOME`变量，变量值为`C:\Program Files\Java\jdk1.7.0_79`  根据自己安装路径填写

    - 新建CLASSPATH变量，变量值为`;%JAVA_HOME%\lib;% JAVA_HOME %\lib\tools.jar`

    - 在path变量(已存在不用新建)添加变量值`%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin` 变量值之间用 **;** 号隔开

  1. 检查是否配置成功

    开始 > 运行 > 输入 javac> 回车，

如下：表示成功

输入 java 如下表示成功

Ant 安装

下载连接:http://ant.apache.org/bindownload.cgi，

下载完成之后解压到你想安装的目录，比如D:\ant。然后将目录添加到系统变量里，

步骤如下：右击我的电脑，点击属性，左侧菜单点击高级系统设置，在弹窗中点击环境变量，然后在用户变量中添加变量ANT_HOME,值为D:\ant，添加变量path，值为D:\ant\bin，

添加变量classpath，值为D:\ant\lib。

配置好之后在终端输入ant -v，如果出现如下图所示则表示成功。

模拟器安装

首先安装 Android SDK http://developer.android.com/sdk/index.html

安装好以后

  1. 打开“环境变量”，在“系统变量”中新建SDK_HOME变量，并将SDK目录添加进去 例如：D:\adt-bundle-windows-x86-sdk
  2. 编辑Path变量。保证现有变量值后面有一个分号 ; ，然后添加%SDK_HOME%\tools;%SDK_HOME%\platform-tools;
  3. SDK环境配置完毕

测试安装是否成功:

在命令行输入android -h

再输入

adb
显示下图表示安装成功

注意：安装硬件加速，需要修改BIOS的virtualization technology为true，模拟器启动不起来，去掉hardware。

创建第一个应用

在命令行进入你想要创建的目录，执行下面的命令

cordova create first com.example.first firstApp

cordova create是创建项目的命令，紧接着first为目录，com.example.first 为命名空间，firstApp为app名称

添加平台

cd first
cordova platform add android

编译app

cordova build android

编译android系统，最后如果看到build successful,就可以了

运行app

cordova emulate android

浏览器上运行

cordova serve android
server at . http://localhost:8000

真机上运行

cordova run android
